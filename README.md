# OmniEmu Website

The official website for [OmniEmu](https://github.com/mileswolfallen2/OmniEmu2.0) — a cross-platform emulator manager, game launcher, and ROM library.

## Pages

- **Home** (`index.html`) — Landing page with features, supported emulators, how it works, and download links
- **Help Guide** (`help-guide.html`) — Installation instructions, setup walkthrough, and troubleshooting
- **FAQ** (`faq.html`) — Frequently asked questions with expandable accordion
- **About** (`about.html`) — Mission, tech stack, contributing, and license info

## Running Locally

No build step required. Just open `index.html` in a browser:

```bash
open index.html
```

Or use a local server:

```bash
npx serve .
```

## Tech

- Vanilla HTML, CSS, JavaScript
- No frameworks, no dependencies
- Dark theme with responsive design
- Scroll animations and FAQ accordion
