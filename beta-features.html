Cloud Sync

Beta

Sync your game saves between devices using Syncthing — free, open-source, and fully local. No third-party accounts or cloud services required.

How It Works

1. Go to Settings > Experimental > Cloud Sync and install Syncthing with one click.

2. Syncthing auto-starts on launch once configured.

3. Host your device or pair with another device to share saves.

4. Manage shared folders and paired devices from the Cloud Sync tab.

5. Access the Syncthing Web UI for advanced configuration.

Privacy-first: All sync happens locally between your devices. No data ever leaves your network unless you configure relay access.

Save Manager

Browse, back up, and delete game saves across all your emulators from one place.

Features

By Game View

Fuzzy-matched titles group saves by game across different emulators, so you can see everything for one game at a glance.

By Emulator View

See all saves organized by which emulator created them, with support for RetroArch's recursive core-directory scanning.

Safety first: OmniEmu backs up your saves before any delete operation, so you can always restore them.

Frontend Support

Beta

Launch alternative frontends directly from OmniEmu for a more console-like experience.

ES-DE — Full integration with auto-configuration. A polished, feature-rich frontend with scraping, themes, and more.
NeoStation — Flutter-based frontend with built-in ScreenScraper integration for automatic game metadata.
Pegasus — Automatic collection file generation from your ROM library. Lightweight and highly customizable.
EmuBuddy — Available to all users (not behind the beta toggle).
How to enable: Go to Settings > Experimental and toggle on Frontend Support.

Decomp Projects

Beta

Native PC ports built from reverse-engineered source code. Bring your own ROM and OmniEmu handles the rest.

Supported Projects

Ship of Harkinian — The Legend of Zelda: Ocarina of Time (N64)
2 Ship 2 Harkinian — The Legend of Zelda: Majora's Mask (N64)
SoH Demo — Ship of Harkinian Demo Build
Portal 64 — Portal (N64 decomp)
Banana Mania PC — Super Monkey Ball Banana Mania PC port

What OmniEmu Does

1. Auto-downloads the latest release from GitHub.

2. Guides you through providing the required ROM file.

3. Manages updates and configuration automatically.

How to enable: Go to Settings > Experimental and toggle on Decomp Projects.

SteamGridDB Cover Art

Search and set custom cover art from SteamGridDB's free API when the automatic scraper doesn't find the right image.

How It Works

1. Open the game detail modal for any game in your library.

2. Click the cover art area and search SteamGridDB.

3. Select a thumbnail — it's downloaded locally for instant display.

4. Use Reset Cover to clear custom art and revert to the auto-scraped version.

Rate-limit aware: OmniEmu automatically retries if the API rate-limits your request, so bulk updates work smoothly.

How to Enable Experimental Features

1. Open OmniEmu and go to Settings.

2. Navigate to Experimental.

3. Toggle on the features you want to try:

Frontend Support — ES-DE, NeoStation, Pegasus, EmuBuddy
Beta Emulators — Additional experimental emulator support
Decomp Projects — Native PC ports from reverse-engineered source
Remote Presets — Fetch recommended config presets from a remote URL
Cloud Sync — Sync saves between devices via Syncthing
Note: Beta features may be unstable. Report any issues on GitHub Issues.
